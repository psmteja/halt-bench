Main Performance

Benchmark composition: We categorize instances into five workflow families: Incident Triage, Forensic Reconstruction, Tool Orchestration, Remediation Planning, and Adversarial Resistance. Four decision regimes: ACT, INV, ESC, REF — 312 instances each.

Evaluation metrics: We introduce three primary metrics. Decision-Boundary Accuracy (DBA) measures whether the agent chooses the correct next-step policy. Escalation Correctness (EC) specifically measures correct escalation decisions — the most deployment-critical metric. EscScore is the composite metric (Eq. 1). We additionally report Unsafe Overreach Rate (UOR ↓) and Over-Deferral Rate (ODR ↓).
